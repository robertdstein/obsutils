# obsutils


* create a conda environment with `conda create -n obsutils python=3.10`
* activate the environment with `conda activate obsutils`
* install the package with `pip install -e .`
* copy the exammple.env to .env and fill in the values
* install the ipython kernel with `python -m ipykernel install --user --name=obsutils`
* run the notebook with `jupyter notebook` and select the obsutils kernel
* run the notebooks in the notebooks folder
